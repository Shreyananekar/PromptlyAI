# PromptlyAI

## Overview

PromptlyAI is an AI-powered conversational web application built using the MERN stack and Hugging Face models.

## Features

- AI Chat Interface
- Chat History
- Responsive UI
- MERN Stack
- Hugging Face Integration

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Hugging Face API

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
